Agora vamos testar alguns comandos de BASH SCRIPTING:

- touch nome-do-arquivo.js -> Cria um arquivo
- rm nome-do-arquivo.js -> Remove um arquivo
- ls -> Lista os arquivos
- mkdir pasta -> Cria uma pasta
- rm -r pasta -> Remove uma pasta e todos os arquivos dentro dela


- Por que isso é importante?

Em muitos momentos você vai se deparar com instruções ou comandos de BASH e é necessário saber o que fazem.
